# matchostat app

## How to get started
- Follow instructions in [Hybrid](https://github.com/martinclsn/hybrid) to install npm, bower and grunt
- ```git clone <matchoui>```
- ```cd matchoui/webapp```
- Only once:
 - ```npm install
bower install```
- ```grunt serve```

## Run in emulator
- ```grunt build```
- ```cd ..```
- ```cordova build <platform>```
- ```cordova emulate <platform>```

Where ```<platform>``` is one of 'ios' and 'android'
