# matchostat app

## How to get started
- Follow instructions [Hybrid](https://github.com/martinclsn/hybrid)
- git clone <matchoui>
- cd matchoui/webapp
- npm install
- bower install
- grunt serve

## Run in emulator
- grunt build
- cd ..
- cordova build <platform>
- cordova emulate <platform>

Where <platform> is one of 'ios' and 'android'
